# ARM Video Player Pro

ARM Video Player Proは、Chromeブラウザ（拡張機能）で動作する高性能なスタンドアロン型ローカルビデオプレイヤーです。
専用タブで動作することで、OSネイティブに近い快適な視聴体験と、Web技術による柔軟な拡張性を両立しています。

![Version](https://img.shields.io/badge/version-0.3.0-blue.svg)

## 🌟 特徴

- **専用タブでの再生**: サイドパネルにあった全画面表示の制限を排除。完全に没入できるフルスクリーン視聴が可能。
- **モダンなUIデザイン**: グラスモーフィズムを採用した美しいOSD（On-Screen Display）による直感的なフィードバック。
- **キーボード操作の完全サポート**: JIS/US配列の両方に対応し、マウスなしでも快適に操作可能。
- **ローカルファイル対応**: ドラッグ＆ドロップやファイル選択から、手持ちの動画ファイルを即座に高画質再生。

## 🎮 操作方法

| 機能 | キーボードショートカット | マウス操作 |
|------|------------------------|------------|
| **再生 / 一時停止** | `Space` | クリック |
| **全画面切り替え** | `F` | ダブルクリック |
| **シーク (±5秒)** | `←` / `→` | - |
| **音量調整 (±10%)** | `↑` / `↓` | - |
| **再生速度 (±0.25x)** | `[` (減速) / `]` (加速) | - |
| **Pic-in-Pic** | `P` | - |

## 🚀 インストール方法

1. このリポジトリをクローンまたはダウンロードします。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をONにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダを選択します。

## 🛠️ 技術スタック

- **Manifest V3**: 最新のChrome Extension仕様に完全準拠。
- **JavaScript (ES Modules)**: モダンな設計による軽量かつ高速な動作。
- **HTML5 Video API**: ブラウザ標準機能を最大限に活用。
- **CSS3 Animations**: スムーズでリッチな視覚効果。

## 📋 今後のロードマップ

- **MKVコンテナ対応**: ブラウザ標準では再生できないMKVファイルのサポート（FFmpeg.wasm予定）。
- **PGS字幕表示**: Blu-rayなどで使用される画像字幕のレンダリング。
- **プレイリスト機能**: 複数の動画を連続再生。
