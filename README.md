# ARM Video Player Pro 🚀

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**ARM Video Player Pro** は、Google Chrome の最新機能（Side Panel API）を活用した、Chromebook 専用の軽量・高速な動画プレイヤー拡張機能です。

## 🌟 プロジェクトの目的
このプロジェクトは、特に **Lenovo Chromebook Plus (MediaTek Kompanio Ultra 910)** のような ARM アーキテクチャを搭載したデバイスにおいて、ブラウザのサイドパネルで動画を効率的に再生・管理することを目的としています。

1Password のオープンソース開発者支援プログラムへの申請を目指し、30日間の継続的な開発を行っています。

## ✨ 特徴
- **Side Panel 統合**: ブラウザの右側に固定されるため、作業を中断せずに動画を視聴可能。
- **ARM 最適化**: MediaTek Kompanio などの ARM プロセッサでスムーズに動作する軽量設計。
- **キーボードフレンドリー**: `FD` のようなクラシックなツールを彷彿とさせる、キーボード主体の操作感。
- **プライバシー配慮**: 外部サーバーを介さず、ローカルファイルを安全にデコードします。

## ⌨️ キーボードショートカット
| キー | 動作 |
| :--- | :--- |
| `Space` | 再生 / 一時停止 |
| `ArrowRight` | 5秒進む |
| `ArrowLeft` | 5秒戻る |
| `F` | フルスクリーン表示 |

## 🛠️ 技術スタック
- Chrome Extension Manifest V3
- Chrome Side Panel API
- HTML5 Video API (Local Object URL)
